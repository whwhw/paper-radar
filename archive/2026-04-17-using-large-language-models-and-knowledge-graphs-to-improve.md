# Using Large Language Models and Knowledge Graphs to Improve the Interpretability of Machine Learning Models in Manufacturing

- **原标题**: Using Large Language Models and Knowledge Graphs to Improve the Interpretability of Machine Learning Models in Manufacturing
- **作者**: Thomas Bayer, Alexander Lohr, Sarah Weiß, Bernd Michelberger, Wolfram Höpken
- **来源**: arXiv
- **发表日期**: 2026-04-17
- **原文**: [https://arxiv.org/abs/2604.16280v1](https://arxiv.org/abs/2604.16280v1)
- **AI 评分**: 8.1 / 10  (论文属于AI核心领域，结合LLM和知识图谱提高ML可解释性，对程序员AI工程和内容创作有直接启发，但摘要稍学术化，需简化解读。)

## 一句话结论
用知识图谱和大语言模型结合，让机器学习结果解释更易懂，提升制造业决策透明度。

## 通俗解读
背景：机器学习结果难解释，影响用户信任。方法：构建知识图谱存储领域数据和ML结果，用大语言模型提取关键信息生成解释。发现：在制造业测试中，能回答复杂问题，提高解释准确性和实用性。意义：理论提出新方法，实践上帮助制造业更好决策，增强AI可解释性。

## 关键方法
构建知识图谱连接领域知识和ML结果，用大语言模型选择性检索生成解释。

## 对你的启发

- **程序员视角**: 可用于AI工程，如自动化工作流中集成知识图谱和LLM，提升系统可解释性，例如在监控工具中自动生成故障报告。
- **投资视角**: 利好AI赛道，特别是可解释AI和制造业自动化，可能推动相关加密项目或长寿科技中的AI应用，关注趋势投资。
- **内容视角**: 抖音可做视频“AI如何像人一样解释决策？”，用制造业案例演示，钩子：揭秘AI黑箱，让普通人也能懂机器学习。
