# Prospective multi-pathogen disease forecasting using autonomous LLM-guided tree search

- **原标题**: Prospective multi-pathogen disease forecasting using autonomous LLM-guided tree search
- **作者**: Sarah Martinson, Michael P. Brenner, Martyna Plomecka, Brian P. Williams, Nicholas G. Reich
- **来源**: arXiv
- **发表日期**: 2026-05-15
- **原文**: [https://arxiv.org/abs/2605.16238v1](https://arxiv.org/abs/2605.16238v1)

## 一句话结论
AI自主开发传染病预测模型，效果不输专家团队。

## 通俗解读
背景：预测传染病传播通常需要专家团队手动建模型，费时费力，难以快速应对新疫情。方法：研究者用大语言模型（如GPT）自动生成预测代码，通过树搜索不断优化，像下棋一样尝试不同策略，选出最佳模型。发现：在2025-2026美国呼吸道季节，AI自动生成的模型组合效果与疾控中心专家团队的黄金标准模型相当甚至更好，尤其在RSV数据稀缺时表现优异。意义：这项技术可大幅降低建模人力成本，快速部署精准疫情预测，未来或能实时预测本地疫情。

## 关键方法
LLM引导的树搜索：让AI不断生成、测试、改进预测模型，自动组合最佳结果。

## 对你的启发

- **程序员视角**: 可在项目中用LLM自动生成和优化算法，比如金融预测或用户行为分析，减少人工调参。
- **投资视角**: AI在医疗预测的自动化能力可能降低行业门槛，利好相关SaaS或数据服务公司；同时注意模型稳健性风险。
